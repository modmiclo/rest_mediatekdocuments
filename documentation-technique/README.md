# Documentation technique API REST

Cette documentation est generee automatiquement depuis les fichiers PHP du dossier `src`.

## Artefacts
- `api-technique.md` : classes et methodes detectees dans les sources API.
- `generate-api-doc.ps1` : script de generation.

## Commande de generation
```powershell
powershell -ExecutionPolicy Bypass -File rest_mediatekdocuments/documentation-technique/generate-api-doc.ps1
```
